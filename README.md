# 📋 Sistema de Cadastro de Pessoas (Java - POO)

## 🚀 Sobre o Projeto

Este projeto é um sistema de cadastro de pessoas desenvolvido em Java utilizando os princípios da Programação Orientada a Objetos (POO).

O sistema permite cadastrar, listar, alterar e excluir pessoas, com suporte para Pessoa Física e Pessoa Jurídica.

---

## 🧠 Conceitos aplicados

* 🔒 Encapsulamento
* 🧬 Herança
* 🎭 Polimorfismo
* 🧩 Abstração

---

## ⚙️ Funcionalidades

* ✅ Cadastro de Pessoa Física
* ✅ Cadastro de Pessoa Jurídica
* ✅ Listagem de pessoas
* ✅ Alteração de dados
* ✅ Exclusão de pessoas
* 🚫 Validação de CPF (não permite duplicados)

---
